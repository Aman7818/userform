# User Form Application

This is a Django project that allows users to submit a form with their details and view the submitted forms.

## Installation

1. Clone the repository:

   ```shell
   git clone https://github.com/Aman7818/userform/
2. Create a virtual environment: python3 -m venv env
source env/bin/activate
3. Install the dependencies: pip install -r requirements.txt
4. Apply database migrations: python manage.py migrate
5. Run the development server: python manage.py runserver
6. Access the application in your browser at http://localhost:8000/user-form/


## Usage

1. Visit the user form page at http://localhost:8000/user-form/
2. Fill in the required details in the form and submit it.
3. The form will be saved, and a confirmation email will be sent to the provided email address.
4. To view all submitted forms, visit http://localhost:8000/form-list/
